# Django Project Repo

This is a sample Django project with CSS and image examples as well as a sample Postgres database creation script which provides a Periodic Table of the Elements.

### Build and Run Project

Simply run the `build_run.sh` script in the root project directory (you may need to set execute permissions on that script with `chmod`)

`./build_run.sh`

CTRL-C to get out of it.

### Run after Build

`python migrate.py runserver`
